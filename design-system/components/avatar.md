# アバター

ユーザーや組織を視覚的に識別するためのコンポーネント。

## Figma
- コンポーネント: https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/Base-Component-Speeda-3.1-MITSUBACHI?node-id=3769-1412
- Do / Don't は Figma ページ内の「仕様」フレームを参照

## 使い分け

### 種類
| 種類 | 用途 |
|------|------|
| **avatar** | 単体で表示する場合 |
| **avatar-group** | 複数のアバターを並べて表示する場合 |

### バリアント
| variant | 使いどころ |
|---------|-----------|
| **image** | ユーザーがプロフィール画像を設定している場合 |
| **text** | 画像がない場合にイニシャルで表示。背景色で個人を区別できる |

### サイズ
| size | 使いどころ |
|------|-----------|
| **small** | リスト項目、コメント欄など省スペースな場所 |
| **medium** | 標準的な表示。ヘッダーやカード内 |
| **large** | やや目立たせたい場所 |
| **x-large** | プロフィール詳細画面など |
| **2x-large** | プロフィールページのメイン表示 |

### 状態
| 状態 | 説明 |
|------|------|
| **通常** | アクティブなユーザー |
| **inactive** | 退職者や非アクティブなアカウントを示す。グレーアウト表示になる |

## アクセシビリティ
- 画像バリアントには `alt` 属性でユーザー名を設定する
- 装飾目的のみの場合は `alt=""` で空にする
- テキストバリアントのイニシャルはスクリーンリーダーで読み上げられるため、必要に応じて `aria-label` でフルネームを補足する

## 並べ方
- 名前やラベルと組み合わせる場合、アバターは左側に配置する
- アバター同士を並べる場合は 4px〜8px の間隔をあける
- 複数人を重ねて表示する場合は、右側のアバターが上に来るようにする
