# タブ

コンテンツを切り替えるナビゲーションコンポーネント。

## Figma
- コンポーネント: https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/Base-Component-Speeda-3.1-MITSUBACHI?node-id=5634-1159
- Do / Don't は Figma ページ内の「仕様」フレームを参照

## コンポーネント一覧

| コンポーネント | 役割 | いつ使うか |
|---------------|------|-----------|
| **page-tab** | ページレベルのナビゲーション | ページ全体でコンテンツを切り替える場合（例：グローバル検索結果、マイスピーダ） |
| **section-tab** | セクションレベルのナビゲーション | ページ内のセクション単位でコンテンツを切り替える場合 |

## 使い分け

### page-tab vs section-tab
| 項目 | page-tab | section-tab |
|------|----------|-------------|
| 用途 | ページレベルの切り替え | セクションレベルの切り替え |
| 折り返し | 不可 | Desktop のみ可 |
| phone サイズ | なし（タップ領域が十分） | あり |

### デバイス別表示（section-tab）
| viewport | 高さ |
|----------|------|
| **desktop** | 32px |
| **phone** | 40px |

### 状態
| state | 説明 |
|-------|------|
| **default** | 通常状態 |
| **hover** | マウスオーバー時 |
| **active** | クリック中 |
| **focus** | キーボードフォーカス時 |
| **disabled** | 操作不可 |

### 選択状態
| selected | 説明 |
|----------|------|
| **true** | 選択中のタブ |
| **false** | 非選択のタブ |

## 振る舞い
- 必ず1つのタブが選択状態になる
- タブをクリックすると対応するコンテンツに切り替わる
- **Desktop**：折り返しを許容（section-tab のみ）
- **Phone**：折り返し不可、横スクロールで対応

### 国・地域の場合（section-tab）
- 国旗（show-flag）を表示して対応する
- 国旗のない地域名（例：ヨーロッパ、アジア）には表示しない
- 国コードや国記号（例：JPN）は使わず、正式名称を表示する

## アクセシビリティ
- `role="tablist"` と `role="tab"` を使用する
- 選択中のタブには `aria-selected="true"` を付ける
- タブとパネルを `aria-controls` で関連付ける
- 矢印キーでタブ間を移動できるようにする
- disabled なタブは `aria-disabled="true"` を付ける

## 並べ方
- page-tab：ページ上部に配置、下線は適用場所によって範囲が異なる
- section-tab：セクションの上部に配置
- タブが多すぎる場合は他のコンポーネント（select-box 等）を検討する
