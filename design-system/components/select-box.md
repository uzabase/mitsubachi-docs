# セレクトボックス

ドロップダウンから選択肢を選ぶコンポーネント。

## 使い分け

### バリアント
| variant | 使いどころ |
|---------|-----------||
| **primary** | 単独で使用する場合。フィルターやソートなど |
| **secondary** | フォーム内で他の入力フィールドと並べる場合 |

### サイズ
| size | 用途 |
|------|------|
| **small** | コンパクトなUI、テーブル内など |
| **medium** | 標準的なフォーム |

### デバイス別表示
| viewport | 説明 |
|----------|------|
| **desktop** | 標準サイズ |
| **phone** | タップしやすいサイズ |

### 状態
| state | 説明 |
|-------|------|
| **default** | 通常状態 |
| **hover** | マウスオーバー時 |
| **active** | クリック中 |
| **focus** | キーボードフォーカス時 |
| **disabled** | 操作不可 |
| **error** | バリデーションエラー時 |

## 振る舞い
- クリックでドロップダウンが開く
- 選択肢をクリックで選択し、ドロップダウンが閉じる
- 選択中の値はトリガー部分に表示される
- プレースホルダーで初期表示のヒントを設定できる
- 選択肢が多い場合はスクロール可能になる

## アクセシビリティ
- `role="combobox"` または `<select>` 要素を使用する
- `aria-expanded` で開閉状態を示す
- `aria-haspopup="listbox"` を付ける
- 矢印キーで選択肢間を移動できるようにする
- Enter で選択、Escape で閉じる
- ラベル（label-unit）と併用する

## 並べ方
- フォーム内では他の入力フィールドと幅を揃える
- ラベルは上または左に配置
- 選択肢の最大表示件数は 8〜10 件程度（それ以上はスクロール）
