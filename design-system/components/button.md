# ボタン

ユーザーのアクションをトリガーするコンポーネント。
すべてのボタンは **ピル型（角が完全に丸い形状）** で統一されている。

---

## 種類と使い分け

### 通常ボタン（Neutral Button）

保存・キャンセルなど、一般的な操作に使う。見た目の強さ（variant）を切り替えて使う。

| variant | 見た目 | 使いどころ |
|---------|--------|-----------|
| **primary** | 黒い塗りつぶし・白文字 | 画面で最も重要なアクション |
| **secondary** | 白背景・黒い枠線 | primary を補助するアクション（キャンセル等） |
| **tertiary** | 白背景・薄い枠線 | さらに控えめなアクション |
| **ghost** | 背景なし・枠線なし | ツールバーやテーブル行内のアクション |
| **plane** | 背景なし・青い文字 | テキストリンクのように見せたいアクション |

### Danger ボタン

削除・解除など、**元に戻せない操作**に使う。常に赤系のスタイルで表示され、ユーザーに注意を促す。

| variant | 見た目 | 使いどころ |
|---------|--------|-----------|
| **primary** | 赤い塗りつぶし・白文字 | 最も強い警告。取り消せない削除など |
| **secondary** | 白背景・赤い枠線・赤文字 | やや控えめな破壊的操作 |
| **ghost** | 背景なし・赤文字 | メニュー内の削除項目など、省スペースな場面 |

### AI ボタン

AI 関連の機能を呼び出すときに使う。専用のスタイルで他のボタンと差別化する。アイコン（✨マーク）は自動的に表示される（指定不要）。

| variant | 見た目 | 使いどころ |
|---------|--------|-----------|
| **primary** | 白背景・グラデーションボーダー（通常時は暗い色合い。ホバーでボーダーが虹色に回転アニメーションする） | AI 機能のメインアクション |
| **secondary** | 白背景・黒い枠線（通常ボタンの secondary に近い見た目。✨アイコンで区別する） | AI 機能の補助的なアクション |

### メニューボタン

ドロップダウンメニューを開くためのボタン。ラベルの右側に下向き矢印（▾）が付いている。クリックすると選択肢のリストが表示される。

| variant | 見た目 | 使いどころ |
|---------|--------|-----------|
| **primary** | 黒い塗りつぶし・白文字・白矢印 | メインのメニュー操作 |
| **secondary** | 白背景・黒い枠線・黒文字・黒矢印 | 補助的なメニュー操作 |
| **ghost** | 背景なし・枠線なし・黒文字・黒矢印 | ツールバーや省スペースなメニュー |

サイズは通常ボタンと同じ 3 段階（medium / large / xLarge）。

### アイコンボタン

テキストなしで、アイコンだけで操作を伝える小さなボタン。閉じる（×）、検索（🔍）、メニュー（⋯）などに使う。

**ラベルを必ず設定する**（アクセシビリティとツールチップのため）。

| variant | 見た目 | 使いどころ |
|---------|--------|-----------|
| **primary** | 濃いグレー背景・白アイコン | 目立たせたい単独のアクション |
| **secondary** | 白背景・黒い枠線 | 枠線で存在感を出したいとき |
| **tertiary** | 白背景・薄い枠線 | secondary より控えめに |
| **ghost**（デフォルト） | 背景なし・枠線なし | ツールバーやテーブル行内など、最もよく使う |

## サイズ

### テキスト付きボタン（通常・Danger・AI 共通）

| size | 使いどころ |
|------|-----------|
| **medium**（デフォルト） | 標準的な UI。フォーム内、ダイアログ内など |
| **large** | やや目立たせたい場面。ページ上部のアクションなど |
| **xLarge** | CTA やヒーロー領域。最も目を引くボタン |

plane variant は size に関わらずコンパクトな見た目になる（テキストリンク風のため）。

### アイコンボタン

| size | 大きさ | 使いどころ |
|------|--------|-----------|
| **small** | 24px 四方 | テーブル行内、タイトル横など省スペースな場所 |
| **medium**（デフォルト） | 32px 四方 | 標準。ツールバー、カード内のアクション |
| **large** | 40px 四方 | ヘッダーやナビゲーションバー |

## 状態

すべてのボタンに共通の状態がある。

| 状態 | どう見えるか |
|------|-------------|
| **通常** | デフォルトの見た目 |
| **ホバー** | マウスを乗せたとき。背景色がわずかに変化する |
| **アクティブ** | クリック中。背景色がさらに濃くなる |
| **フォーカス** | キーボード操作時。黒い二重リングが表示される |
| **無効（disabled）** | 操作できない。色が薄くなり、カーソルが禁止マークに変わる |
| **ローディング（loading）** | 処理中。スピナーが表示され、クリックできなくなる |

### 選択状態（selected）

トグル操作で「今これが選ばれている」ことを示す状態。選択されると**青い背景**に変わる。
タブの切り替え、フィルター、ブックマークのオン/オフなどに使う。

**通常ボタンで selected が使える variant:**

| variant | selected | selected + disabled | selected + loading |
|---------|:--------:|:-------------------:|:------------------:|
| **primary** | — | — | ✅ |
| **secondary** | ✅ | ✅ | ✅ |
| **tertiary** | ✅ | ✅ | ✅ |
| **ghost** | ✅ | ✅ | ✅ |
| **plane** | — | — | — |

- secondary / tertiary / ghost は、選択状態のまま disabled にしたり loading にしたりできる（例: 選択中のフィルターを一時的に無効化）
- primary は loading 中のみ選択背景が維持される。通常時の選択スタイルはない
- plane には選択状態がない

**アイコンボタン**も secondary / tertiary / ghost で選択状態が使える。

**Danger ボタン** / **AI ボタン** には選択状態そのものがない。

---

## 並べ方

- **primary と secondary を横に並べる場合**: primary を右に、secondary を左に配置する
- **danger ボタン**: 確認ステップ（ダイアログ等）を挟んでから実行する構成が望ましい
- ボタン同士の間隔は **8px** 程度

---

## Do / Don't

### ✅ Do

- **primary ボタンは 1 画面に 1 つ** — 最も目立つボタンなので、画面の中心的なアクションだけに使う。補助的なアクションには secondary や ghost を使う
- **danger ボタンの前に確認ステップを入れる** — 削除などの操作はすぐに実行せず、ダイアログ等で「本当にいいですか？」を挟む
- **アイコンボタンにはラベルを付ける** — テキストがないボタンにはスクリーンリーダー用のラベルを必ず設定する。ツールチップとしても表示されるので、目の見えるユーザーにとっても有益
- **選択状態は secondary / tertiary / ghost で使う** — タブやフィルターの切り替えで「今選ばれている」ことを示すのに使う

### ❌ Don't

- **primary や plane を「選択状態」にしない** — 選択スタイルが存在しない。付けてもデフォルトの見た目のまま変わらない
- **Danger ボタンや AI ボタンを「選択状態」にしない** — これらにも選択スタイルは用意されていない
- **AI ボタンにアイコンを指定しない** — AI ボタンのアイコン（✨）は自動で表示される。指定しても無視される
- **ボタンのラベルを長文にしない** — 「保存する」「削除する」のような短い動詞で書く。長い文章はボタンの外に書く



## デザイン仕様（Neutral Button）

### 共通

- 形状: **ピル型**（`border-radius: 9999px`）
- アイコンとテキストの間隔: `4px`
- line-height: `1.5`
- letter-spacing: フォントサイズの約 1%（12px → 0.12px、14px → 0.14px、16px → 0.16px）
- フォーカスリング: `box-shadow: 0 0 0 2px #fff, 0 0 0 4px #191919`

### サイズ別の寸法

| size | 最小高さ | 左右パディング | フォントサイズ | アイコンサイズ |
|------|---------|--------------|-------------|-------------|
| **medium** | 32px | 12px | 12px | 18px |
| **large** | 40px | 16px | 14px | 20px |
| **xLarge** | 48px | 16px | 16px | 22px |

plane variant は左右パディングが `4px` で、最小高さの制約がない（テキストリンク風）。

### Variant 別のスタイル（通常状態）

| variant | 背景 | ボーダー | テキスト色 | font-weight |
|---------|------|--------|----------|-------------|
| **primary** | `#282828` | なし | `#ffffff` | bold (700) |
| **secondary** | 透明 | `rgba(0,0,0,0.84)` 1px solid | `rgba(0,0,0,0.84)` | bold (700) |
| **tertiary** | 透明 | `rgba(0,0,0,0.2)` 1px solid | `rgba(0,0,0,0.84)` | regular (400) |
| **ghost** | 透明 | なし | `rgba(0,0,0,0.84)` | regular (400) |
| **plane** | 透明 | なし | `#315ce8`（リンク青） | regular (400) |

### 状態別の色変化（primary）

| 状態 | 背景 | テキスト色 |
|------|------|----------|
| **通常** | `#282828` | `#ffffff` |
| **ホバー** | `#191919` | `#ffffff` |
| **アクティブ** | `#000000` | `#ffffff` |
| **フォーカス** | `#282828` + フォーカスリング | `#ffffff` |
| **無効** | `rgba(0,0,0,0.07)` | `rgba(0,0,0,0.35)` |
| **ローディング** | `rgba(0,0,0,0.07)` | `rgba(0,0,0,0.84)` |

### 状態別の色変化（secondary / tertiary / ghost）

| 状態 | 背景 |
|------|------|
| **ホバー** | `rgba(0,0,0,0.07)` |
| **無効** | `rgba(0,0,0,0.03)` + ボーダー `rgba(0,0,0,0.07)` |

テキスト色・ボーダー色は通常状態と同じ。無効時はテキスト色が `rgba(0,0,0,0.35)` に変わる。

### 選択状態の色

| 状態 | 背景 | ボーダー（secondary） | テキスト色（secondary） |
|------|------|---------------------|---------------------|
| **選択 通常** | `#e8edff` | `#3f69f2` | `#315ce8` |
| **選択 ホバー** | `#dbe4ff` | `#3f69f2` | `#315ce8` |
| **選択 アクティブ** | `#d5dfff` | `#3f69f2` | `#315ce8` |

tertiary / ghost の選択状態は背景が同じ `#e8edff` 系だが、ボーダーとテキスト色は通常状態のまま。

## デザイン仕様（AI Button）

サイズ別の寸法（最小高さ、パディング、フォントサイズ、アイコンサイズ）は Neutral Button と同一。
font-weight は primary / secondary ともに **bold (700)**。

AI ボタンには ✨（magic-fill）アイコンが常に自動表示される。

### primary の特徴

- large / xLarge には最小幅 `80px` が設定されている
- 色は Neutral Button の primary と同じ（`#282828` → `#191919` → `#000000`）
- ホバー時にボーダーがグラデーション（虹色）の回転アニメーションになる（Figma 上は static だが実装にはアニメーションあり）

### secondary の特徴

- Neutral Button の secondary と同じ配色（ボーダー `rgba(0,0,0,0.84)` + テキスト `rgba(0,0,0,0.84)`）
- アクティブ時の背景: `rgba(0,0,0,0.1)`（Neutral より若干濃い）

### 状態別の色

primary / secondary ともに Neutral Button と同じ色体系を使う。

| 状態 | primary 背景 | secondary 背景 |
|------|------------|--------------|
| **通常** | `#282828` | 透明 |
| **ホバー** | `#191919` | `rgba(0,0,0,0.07)` |
| **アクティブ** | `#000000` | `rgba(0,0,0,0.1)` |
| **フォーカス** | `#282828` + フォーカスリング | 透明 + フォーカスリング |
| **無効** | `rgba(0,0,0,0.07)` | ボーダー `rgba(0,0,0,0.07)` |
| **ローディング** | `rgba(0,0,0,0.07)` | ボーダー `rgba(0,0,0,0.07)` |

無効・ローディング時のテキスト色:
- 無効: `rgba(0,0,0,0.35)`
- ローディング: `rgba(0,0,0,0.84)`

## デザイン仕様（Danger Button）

サイズ別の寸法（最小高さ、パディング、フォントサイズ、アイコンサイズ）は Neutral Button と同一。

### Variant 別のスタイル（通常状態）

| variant | 背景 | ボーダー | テキスト色 | font-weight |
|---------|------|--------|----------|-------------|
| **primary** | `#db351f`（赤） | なし | `#ffffff` | bold (700) |
| **secondary** | 透明 | `#db351f` 1px solid | `#c92812`（濃い赤） | bold (700) |
| **ghost** | 透明 | なし | `#c92812`（濃い赤） | regular (400) |

### 状態別の色変化（primary）

| 状態 | 背景 | テキスト色 |
|------|------|----------|
| **通常** | `#db351f` | `#ffffff` |
| **ホバー** | `#c92812` | `#ffffff` |
| **アクティブ** | `#b02412` | `#ffffff` |
| **フォーカス** | `#db351f` + フォーカスリング | `#ffffff` |
| **無効** | `rgba(0,0,0,0.07)` | `rgba(0,0,0,0.35)` |
| **ローディング** | `rgba(0,0,0,0.07)` | `rgba(0,0,0,0.84)` |

### 状態別の色変化（secondary / ghost）

| 状態 | 背景 |
|------|------|
| **ホバー** | `#ffedeb`（淡い赤） |
| **アクティブ** | `#ffebe8`（やや濃い淡い赤） |

secondary はボーダー `#db351f` を維持。ghost はボーダーなし。
無効時はテキスト色 `rgba(0,0,0,0.35)`、secondary のボーダーは `rgba(0,0,0,0.07)` に変わる。

## デザイン仕様（Menu Button）

ラベルの右側に下向き矢印（`chevron-down-small`）が付くボタン。クリックでドロップダウンメニューを開く。
「操作の入り口」として使い、現在の値の表示・選択には使わない（それは select-box を使う）。

### Neutral Button との違い

- **左右のパディングが非対称**: 左は通常、右はシェブロン分だけ狭い（`8px`）
- **シェブロンアイコン**が自動で付く（サイズはアイコンと同じ）

### サイズ別の寸法

| size | 最小高さ | 左パディング | 右パディング | フォントサイズ | シェブロンサイズ | gap |
|------|---------|------------|------------|-------------|-------------|-----|
| **medium** | 32px | 12px | 8px | 12px | 18px | 2px |
| **large** | 40px | 16px | 8px | 14px | 20px | 4px |
| **xLarge** | 48px | 16px | 8px | 16px | 22px | 4px |

### Variant 別のスタイル

配色・font-weight・状態変化はすべて Neutral Button の primary / secondary / ghost と同一。

## デザイン仕様（Icon Button）

テキストを持たず、アイコンだけで操作を伝える正円のボタン。
必ずツールチップ（ラベル）で操作の意味を補完する。

### サイズ別の寸法

| size | ボタンサイズ（幅=高さ） | アイコンサイズ | パディング |
|------|---------------------|-------------|----------|
| **small** | 24px | 18px | 2px |
| **medium** | 32px | 20px | 4px |
| **large** | 40px | 22px | 4px |

形状は `border-radius: 9999px`（正円）。

### Variant 別のスタイル（通常状態）

| variant | 背景 | ボーダー | アイコン色 |
|---------|------|--------|----------|
| **primary** | `#282828` | なし | `#ffffff` |
| **secondary** | 透明 | `rgba(0,0,0,0.84)` 1px solid | `rgba(0,0,0,0.84)` |
| **tertiary** | 透明 | `rgba(0,0,0,0.2)` 1px solid | `rgba(0,0,0,0.84)` |
| **ghost**（デフォルト） | 透明 | なし | `rgba(0,0,0,0.84)` |

### 選択状態の色

| variant | 背景 | ボーダー |
|---------|------|--------|
| **secondary** | `#e8edff` | `#3f69f2` |
| **tertiary** | `#e8edff` | `rgba(0,0,0,0.1)` |
| **ghost** | `#e8edff` | なし |

状態変化（ホバー・アクティブ・フォーカス・無効）の色は Neutral Button と同一。
