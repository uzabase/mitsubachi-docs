# レポート見出し

レポートやドキュメント内のセクション見出しを表示するコンポーネント。

## Figma
- コンポーネント: https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/Base-Component-Speeda-3.1-MITSUBACHI?node-id=9479-1540
- Do / Don't は Figma ページ内の「仕様」フレームを参照

## 使い分け

### レベル
| level | 用途 |
|-------|------|
| **1** | ページ全体の主タイトル（h1 相当） |
| **2** | 大きなセクションの見出し（h2 相当） |
| **3** | サブセクションの見出し（h3 相当）。左に縦線のアクセントあり |
| **4** | 小見出し（h4 相当） |
| **5** | 詳細項目の見出し（h5 相当） |
| **6** | 最小の見出し（h6 相当） |

### デバイス別表示
| viewport | 説明 |
|----------|------|
| **desktop** | 標準サイズで表示 |
| **phone** | コンパクトなサイズで表示 |

### アクションスロット
- 見出しの右側にボタンやリンクを配置できる
- その見出しレベルを包含するコンテンツに対するアクションや参照情報を表示

## 振る舞い
- 見出しレベルは HTML の h1〜h6 に対応させる
- 折り畳み/展開機能は現時点では提供しない（将来検討予定）

## アクセシビリティ
- 適切な見出しレベル（h1〜h6）を使用し、階層構造を正しく表現する
- 見出しレベルを飛ばさない（h2 の次に h4 を使わない）
- アクションボタンには適切な `aria-label` を付ける

## 並べ方
- 見出しとコンテンツの間には適切な余白を設ける
- 同じレベルの見出しは同じスタイルで統一する
- アクションスロットのボタンは見出しテキストの右側に配置
