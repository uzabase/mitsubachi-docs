# タグ

情報にラベルを付けて分類・識別するためのコンポーネント。

## Figma
- コンポーネント: https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/Base-Component-Speeda-3.1-MITSUBACHI?node-id=5305-1126
- Do / Don't は Figma ページ内の「仕様」フレームを参照

## コンポーネント一覧

| コンポーネント | 役割 | いつ使うか |
|---------------|------|-----------|
| **link-tag** | クリック可能なタグ | タグをクリックして絞り込み・遷移する場合 |
| **read-only-tag** | 読み取り専用のタグ | 情報の表示のみで操作が不要な場合 |

## 使い分け

### コンポーネントの選択
- ユーザーがタグを操作する → `link-tag`
- 情報を表示するだけ → `read-only-tag`

### パターン
| パターン | 用途 |
|----------|------|
| **label** | 基本的なラベル表示 |
| **stiff** | 強調が必要なラベル |
| **outlined** | 軽い強調、背景色なしで表示したい場合 |

### サイズ
| サイズ | 用途 |
|--------|------|
| **XS** | 最小サイズ、スペースが限られる場所 |
| **S** | 小さめのタグ |
| **M** | 標準サイズ |
| **L** | 大きめの表示が必要な場合 |

### 背景色（color）
- 情報の種類やカテゴリに応じて色を使い分ける
- 色だけで情報を伝えず、テキストと併用する

## 状態
| state | 説明 |
|-------|------|
| **default** | 通常状態 |
| **hover** | マウスオーバー時（インタラクティブな場合） |

## 使用例
- 企業情報の属性表示（業種、地域など）
- フィルター条件の表示
- ステータス表示
- カテゴリ分類

## アクセシビリティ
- 色だけで情報を伝えない（テキストを併用）
- クリック可能なタグには適切なフォーカス表示を付ける

## 並べ方
- 複数のタグは横に並べる（8px 間隔）
- 折り返しが発生する場合は行間も 8px
