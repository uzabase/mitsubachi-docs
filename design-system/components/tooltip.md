# ツールチップ

UI 要素の補足情報をホバー時に一時表示するコンポーネント。

## Figma
- URL: https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/Base-Component-Speeda-3.1-MITSUBACHI?node-id=6656-2365
- バリアント・サイズ・色・寸法の詳細は Figma を参照

## Do / Don't

### ✅ Do
- アイコンボタンなど意味が伝わりにくい要素に付ける
- テキストは短く簡潔にする
- ホバーで表示・マウスアウトで非表示にする

### ❌ Don't
- 操作に必須の情報をツールチップだけに入れない
- ツールチップ内にリンクやボタンなどのインタラクティブ要素を入れない
- ラベルの代わりに使わない（ラベルは常に見えるべき）
- 長文を入れない（最大幅 200px を超える量は避ける）

## 並べ方
- トリガー要素の近くに表示する（重ならないよう配置を自動調整）
- 他の UI を大きく遮らない位置にする
