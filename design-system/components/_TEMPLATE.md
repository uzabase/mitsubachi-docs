# コンポーネントドキュメント テンプレート

> 新しいコンポーネントのドキュメントを書くときは、このテンプレートをコピーしてください。
>
> **書く前に確認すること:**
> - Figma のコンポーネントフレームを見て、variant / size / state / selected などのプロパティを洗い出す
> - Storybook も確認し、Do/Don't のヒントを探す
>
> **書き方の方針:**
> - 読者は「このコンポーネントを使って画面を作りたい人」
> - バリアント・サイズ・色・寸法の詳細は Figma に委ねる。ドキュメントには使い分けのルールと配置ルールを書く
> - 該当しないセクションは削除してよい
> - `{...}` の部分を実際の内容に置き換える
