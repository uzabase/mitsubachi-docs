# ラベルユニット

入力フィールドのラベル、必須表示、補足テキストをまとめたコンポーネント。

## Figma
- コンポーネント: https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/Base-Component-Speeda-3.1-MITSUBACHI?node-id=3-327
- Do / Don't は Figma ページ内の「仕様」フレームを参照

## 使い分け

### 構成要素
| 要素 | 必須/任意 | 説明 |
|------|:--------:|------|
| **ラベル** | 必須 | 入力内容を示すテキスト |
| **必須バッジ** | 任意 | 入力が必須であることを示す |
| **サポートテキスト** | 任意 | 入力のヒントや補足説明 |

### パターン
| パターン | 使いどころ |
|---------|-----------|
| **ラベルのみ** | シンプルな入力項目 |
| **ラベル + 必須** | 必須入力の項目 |
| **ラベル + サポートテキスト** | 入力形式の説明が必要な場合 |
| **ラベル + 必須 + サポートテキスト** | 必須かつ補足説明が必要な場合 |

## アクセシビリティ
- ラベルは `<label>` 要素で入力フィールドと関連付ける
- 必須項目は `required` 属性を付け、視覚的にも「必須」と明示する
- サポートテキストは `aria-describedby` で入力フィールドと紐付ける
- プレースホルダーをラベルの代わりにしない

## 並べ方
- ラベルは入力フィールドの上に配置
- 必須バッジはラベルの右隣に配置
- サポートテキストはラベルの下、入力フィールドの上に配置
- ラベルと入力フィールドの間隔は 8px
