# 検索ボックス

テキストを入力してデータを検索するコンポーネント。

## Figma
- コンポーネント: https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/Base-Component-Speeda-3.1-MITSUBACHI?node-id=5789-1881
- Do / Don't は Figma ページ内の「仕様」フレームを参照

## 使い分け

### バリアント
| variant | 使いどころ |
|---------|-----------|
| **primary** | 検索がページ上での中心的な役割で、単独で使用する場合。テーブルやリストのデータ検索など |
| **secondary** | フォームに包含される場合。text-field や select-box と一緒に使用される場所 |

### デバイス別表示
| viewport | 説明 |
|----------|------|
| **desktop** | 標準サイズ |
| **phone** | コンパクトなサイズ |

## 振る舞い
- 入力中はクリアボタン（×）が表示される
- クリアボタンで入力した文字を一括消去できる
- 検索中はローディング状態を表示できる
- プレースホルダーで検索対象のヒントを表示できる

## アクセシビリティ
- `type="search"` または `role="searchbox"` を使用する
- ラベル（label-unit）と併用して「何を」検索するのかを明示するのが望ましい
- ラベルの併用が難しい場合は、プレースホルダーで補足する
- クリアボタンには `aria-label="検索テキストをクリア"` などを付ける

## 並べ方
- テーブルやリストの上に配置する
- 他のフィルター要素と横に並べる場合は左側に配置
- 幅は検索対象に応じて調整（短すぎると入力しづらい）
