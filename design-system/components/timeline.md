# タイムライン

時系列の出来事やステップを視覚的に表示するコンポーネント。

## Figma
- コンポーネント: https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/Base-Component-Speeda-3.1-MITSUBACHI?node-id=6931-5681
- Do / Don't は Figma ページ内の「仕様」フレームを参照

## 使い分け

### パターン
| パターン | 用途 |
|----------|------|
| **Normal** | 標準的なタイムライン表示 |
| **Loose** | 余白を多めにとった表示 |

### timeline-item の位置
| 位置 | 説明 |
|------|------|
| **First** | 最初のアイテム |
| **Middle** | 中間のアイテム |
| **Last** | 最後のアイテム |
| **Both** | 最初かつ最後（1件のみの場合） |

### 強調（emphasized）
- 重要なイベントを強調表示できる
- 主要な意思決定や転機（買収など）に使用
- 現状のステータスを示す場合にも使用

## 使用例
- 企業の沿革・歴史表示
- 経営理念テーマの変遷
- プロジェクトのマイルストーン
- ニュースや発表の時系列表示

## 振る舞い
- 縦方向に時系列で並ぶ
- 線でアイテム間を接続
- 各アイテムには日付とコンテンツを表示

## アクセシビリティ
- リスト構造（`<ol>` または `<ul>`）でマークアップする
- 時系列の順序が明確になるようにする
- 強調アイテムは視覚以外でも区別できるようにする

## 並べ方
- 基本は上から下へ時系列順
- 最新が上か、古い順が上かは用途に応じて決める
