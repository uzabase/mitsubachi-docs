# ページネーション

テーブルやリストのページ切り替えを行うコンポーネント。

## Figma
- コンポーネント: https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/Base-Component-Speeda-3.1-MITSUBACHI?node-id=8909-6683
- Do / Don't は Figma ページ内の「仕様」フレームを参照

## 使い分け

### デバイス別表示
| viewport | 配置 |
|----------|------|
| **desktop** | リストの下に配置。データが多い場合は上部にも追加可 |
| **phone** | リストの下のみに配置（上には配置しない）。中央揃え |

### 構成要素
| 要素 | 説明 |
|------|------|
| **前のページボタン** | 前のページの先頭に遷移（icon-button） |
| **ページ番号** | 現在のページ番号を表示・選択（select-box） |
| **ページ数** | 全体のページ数を表示 |
| **次のページボタン** | 次のページの先頭に遷移（icon-button） |

## 振る舞い
- 先頭ページ表示中は「前のページ」ボタンが disabled になる
- 最終ページ表示中は「次のページ」ボタンが disabled になる
- ページ番号の select-box から任意のページに直接遷移できる
- ページ遷移後は、テーブルやリストの先頭行を表示する

## アクセシビリティ
- ページ切り替えボタンには `aria-label` で「前のページ」「次のページ」を付ける
- 現在のページ番号をスクリーンリーダーに伝える
- disabled 状態のボタンは `aria-disabled="true"` を付ける

## 並べ方
- PC: テーブルやリストの下に左揃えで配置
- PC: データ量が多くスクロールが長い場合は、上部にも追加で配置できる
- モバイル: リストの下のみ、中央揃えで配置
