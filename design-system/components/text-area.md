# テキストエリア

複数行のテキスト入力を受け付けるフォームコンポーネント。

## Figma
- コンポーネント: https://www.figma.com/design/kHQNLM1dnk0EhZwOKBEBkL/Base-Component-Speeda-3.1-MITSUBACHI?node-id=3-326
- Do / Don't は Figma ページ内の「仕様」フレームを参照

## 使い分け

### text-field との違い
| 項目 | text-field | text-area |
|------|------------|-----------|
| 行数 | 1行 | 複数行 |
| 用途 | 名前、メールなど短いテキスト | コメント、説明文など長いテキスト |
| リサイズ | 不可 | ドラッグでリサイズ可能 |

## 状態
| state | 説明 |
|-------|------|
| **default** | 通常状態 |
| **hover** | マウスオーバー時 |
| **focus** | フォーカス時（青い枠線） |
| **error** | エラー時（赤い枠線） |
| **disabled** | 操作不可 |

## 振る舞い

### ドラッグでリサイズ
- ユーザーが右下をドラッグして高さを変更可能
- 横幅は固定（親要素に従う）

### 入力エリアの高さ
- 初期表示は用途に応じた行数を確保する
- 最小高さを設定し、小さくなりすぎないようにする

### 文字数カウント
- 必要に応じて「85/100」のように現在の文字数/最大文字数を表示
- 最大文字数に近づいたら警告色で表示

## アクセシビリティ
- `<label>` 要素と関連付ける
- エラー時は `aria-invalid="true"` と `aria-describedby` でエラーメッセージを関連付ける
- プレースホルダーをラベルの代わりにしない

## 並べ方
- ラベルはテキストエリアの上に配置
- エラーメッセージはテキストエリアの下に配置
- 文字数カウントは右下に配置
