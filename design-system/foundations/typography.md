## Primitive Type Scale

| 値 | 命名 | サイズ | 対応アイコンサイズ |
|----|------|--------|------------------|
| 10px | primitive-type-scale-1 | 2x-small | 14px |
| 11px | primitive-type-scale-2 | x-small | 16px |
| 12px | primitive-type-scale-3 | small | 18px |
| 14px | primitive-type-scale-4 | medium | 20px |
| 16px | primitive-type-scale-5 | large | 22px |
| 18px | primitive-type-scale-6 | x-large | 24px |
| 20px | primitive-type-scale-7 | 2x-large | 26px |
| 22px | primitive-type-scale-8 | 3x-large | 28px |

## Font Weight

| ウェイト | 用途 |
|---------|------|
| **regular** | 本文、通常のテキスト |
| **bold** | 見出し、強調 |

## 使い分けの目安

| サイズ | 用途 |
|--------|------|
| **10px〜12px** | キャプション、注釈、補足テキスト |
| **14px** | 本文（標準） |
| **16px〜18px** | 小見出し、強調テキスト |
| **20px〜25px** | セクション見出し |
| **28px以上** | ページタイトル、ヒーロー |
