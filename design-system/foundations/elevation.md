## ステータス

> **検討中**: シャドウとグラデーションの定義は現在整理中です。

## 検討メモ

- シャドウと一緒にグラデーションを考える
- グラデーションはナビなどの「潜り込む」表現に使用
- FAB（Floating Action Button）は検討中

## 現在の方針

### シャドウの用途
| 用途 | 説明 |
|------|------|
| **カード** | 背景から浮き上がる表現 |
| **ドロップダウン** | メニューやセレクトボックスの展開時 |
| **モーダル** | ダイアログの背景との分離 |
| **FAB** | フローティングアクションボタン（検討中） |

### ライト/ダークテーマ
- ライトテーマとダークテーマで影の見え方が異なる
- 詳細は Figma を参照

## 注意点
- 影を多用しすぎない（階層が曖昧になる）
- 影のレベルは要素の重なり順に対応させる
